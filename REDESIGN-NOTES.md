# RielArt 2027 redesign notes

This package keeps the complete approved RielArt website content and information architecture while replacing the presentation layer with a new premium liquid-glass design system.

## Design direction

- Editorial scale and confidence without exaggerated claims
- Conversion-led page hierarchy and strong calls to action
- Restrained glass surfaces, depth, blur, and blue/cyan light
- Light and dark themes with the existing persistent theme toggle
- Consistent visual language with the RielArt Client Portal
- Responsive, accessible, reduced-motion-friendly interactions

## Deployment

Upload the complete contents of this folder to the existing `live-rielart` GitHub repository. GitHub Pages will continue using the existing `CNAME` file and custom-domain configuration.

No DNS, Stripe, Formspree, or Cloudflare changes are required for this redesign.
